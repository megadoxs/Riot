# Riot 
Riot is a website that lets users explore all [League of Legends](https://www.leagueoflegends.com/) champions, including their abilities and available skins.<br> 
It also provides detailed information on all [Valorant](https://playvalorant.com/) agents and their abilities.

## Website
You can access the website here: https://megadoxs.github.io/Riot/

## Features

  - Always up to date with the latest game data
  - Complete list of all League of Legends champions
  - Detailed champion abilities and descriptions
  - Full gallery of available champion skins
  - Complete list of all Valorant agents
  - Detailed agent abilities and descriptions
